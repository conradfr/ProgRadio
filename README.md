ProgRadio
=========

http://www.programmes-radio.io

Dependencies
--------------
- VueJs2
- PHP 7.1 / Symfony 3.3
- NodeJS
- Elixir 1.5+
- PosgreSQL / Redis
- Nginx or Apache
- Capistrano (deployment)

Run
--------------
**Scraper**: cd Scraper && node index.js

**Daemon**: cd Importer && iex -S mix (dev)
